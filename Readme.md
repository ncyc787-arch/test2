# iMessage for SillyTavern

Плавающая иконка-телефон с iMessage-интерфейсом для переписки с активным персонажем ST. Переписка привязана к каждому чату отдельно (per-chat через `chat_metadata`).

## Установка
1. Папка лежит в `public/scripts/extensions/third-party/iMessage/`.
2. F5 → Extensions → Manage extensions → включи iMessage.
3. Справа на экране появится синяя иконка — зажми и тащи куда удобно, короткий тап открывает приложение.

## Что делает
- Отдельная переписка с персонажем, имя и аватар берутся из текущего чата ST (per-chat).
- **Двусторонняя синхронизация с основным чатом**:
  - Вся переписка из телефона подмешивается в промпт основного чата — бот знает о чём вы говорили.
  - Если в основном чате бот пишет «написал в телефон: "..."» — текст автоматически появится в расширении (без РП-действий).
- **Авто-сообщения**: включи в настройках — персонаж сам напишет спустя случайный интервал.
- **Картинки**: персонаж может сам прислать фото тегом `[IMG:...]`, ты — через скрепку. Генерация через свой OpenAI/Gemini endpoint или настройки sillyimages (если не задано своё).
- **Удалённые сообщения**: ✕ рядом со своим сообщением → зачёркнутое. Второй раз ✕ → убрать совсем.
- **Vision**: если бот поддерживает картинки, он их «видит» и помнит через авто-caption.
- **Стиль письма**: при первом ответе бота генерится метаданные (как часто пишет, объём, эмодзи) по описанию персонажа — кэшируется. Перегенерить в настройках.

## Настройки (шестерёнка в шапке)
- **LLM API** — свой endpoint, ключ, модель. Все запросы идут сюда, основной API ST не трогается.
- **Image API** — свой OpenAI/Gemini или наследуется из sillyimages.
- **Промпты** — префикс/суффикс/negative, которые добавляются к каждой генерации фото.
- **Синк**: инжектить ли переписку в основной чат, вытаскивать ли «написал в телефон» из РП.
- **Авто-сообщения**: мин/макс минуты интервала.

## Консоль
```js
imsgOpen()     // открыть
imsgDebug()    // состояние текущего чата
imsgInject()   // посмотреть что идёт в промпт основного чата
imsgReset()    // стереть переписку в текущем чате
imsgFabReset() // вернуть иконку на дефолтную позицию
```

## Файлы
- `manifest.json` — мета
- `index.js` — точка входа, FAB, события
- `ui.js` — рендер чата и настроек
- `engine.js` — генерация ответов, парсинг РП, двусторонний синк
- `api.js` — независимый LLM и image клиент (+ Gemini, /sd fallback, NSFW-санитайзер)
- `state.js` — per-chat `chat_metadata`, глобальные `extension_settings`
- `style.css` — стили
